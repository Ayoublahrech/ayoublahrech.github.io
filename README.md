## 📋 Description
Une application de conversion d'unités ultra-rapide, légère et conçue pour fonctionner **100% hors-ligne**. 
Développée sous la bannière **Andel Slim**, cette application permet de convertir des longueurs, poids, températures et bien plus en un seul clic, sans aucune latence.

## 🚀 Caractéristiques
- **Offline First** : Aucune connexion internet requise après installation.
- **Privacy Focused** : Respect total de la vie privée, aucune donnée collectée.
- **Fast UI** : Interface minimaliste optimisée pour la rapidité.
- **Global Ready** : Support des systèmes métriques et impériaux.
